  <h1>OTT Clone App (Netflix)</h1>

  <h2>Project Overview</h2>
  <p>
    This is a Netflix-inspired OTT platform clone built to demonstrate full-stack development skills, including UI development, backend integration, and database management.
  </p>

  <h2>Features</h2>
  <ul>
    <li>User authentication and authorization</li>
    <li>Video streaming and playback</li>
    <li>Responsive and interactive UI with React.js</li>
    <li>Backend services powered by Node.js and Java</li>
    <li>Data storage with MongoDB</li>
  </ul>

  <h2>Technologies Used</h2>
  <ul>
    <li><strong>Frontend:</strong> JavaScript, React.js, HTML, CSS</li>
    <li><strong>Backend:</strong> Node.js, Java</li>
    <li><strong>Database:</strong> MongoDB</li>
  </ul>

  <h2>Setup Instructions</h2>
  <ol>
    <li>Clone the repository to your local machine.</li>
    <li>Install dependencies for both frontend and backend using <code>npm install</code>.</li>
    <li>Configure your MongoDB connection in the backend.</li>
    <li>Run the frontend and backend servers.</li>
  </ol>
